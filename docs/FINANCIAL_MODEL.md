# Financial Model (Per Customer)

## Baseline Operating Cost (Launch)

- Supabase Pro: $25/mo
- AWS EC2 Spot: $18/mo
- Cloudflare R2: $5/mo
- Marketing tools (amortized): $150/mo
- Total OPEX: $198/mo

## Example Unit Economics

Assume customer with $1M MRR:

- Recovered revenue lift (0.5%): $5,000/mo
- Syzm platform fee: $1,000/mo
- Syzm success fee (5% of recovered): $250/mo
- Gross revenue to Syzm: $1,250/mo

Projected net margin at launch assumptions: ~84.1%.
