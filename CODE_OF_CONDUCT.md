# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Standards

- Be respectful and specific in technical critique.
- Assume good intent and focus on solving the issue.
- Keep discussions factual, reproducible, and inclusive.

## Unacceptable Behavior

- Harassment, discriminatory language, or personal attacks
- Doxxing or sharing private data without consent
- Repeated unconstructive disruption

## Enforcement

Project maintainers may remove comments, commits, or access for violations.

To report behavior concerns, contact `conduct@syzm.ai`.
